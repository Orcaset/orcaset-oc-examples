# Orcaset Examples

Example financial models using Orcaset in OCaml.

## Example Projects

- **[basic_statement](basic_statement/)**: A basic financial statement model demonstrating core Orcaset features.

## Getting Started

To run the examples, ensure you have OCaml installed. Clone the repository and navigate to the desired example directory. Follow the instructions in each example's README for setup and execution.

If you do not have OCaml installed, it is recommended to install it via [OPAM](https://opam.ocaml.org/doc/Install.html), the official OCaml package manager. 

Use your favorite coding agent to help setup an environment and run the project if you receive unexpected errors.
